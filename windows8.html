Windows 8
2011
CREATOR
Microsoft
ABOUT

Windows 8 is a personal computer operating system developed by Microsoft. It introduced major changes to the operating system’s platform and user interface to improve its user experience on tablets. In particular, it included a touch-optimized Windows shell. Although it had positive reactions towards performance improvements, security enhancements, and improved support for touch screen devices, it was widely criticized for being confusing and difficult to learn.

CREATOR
Microsoft
HOW TO USE THE SCROLLBAR

Similar to modern scrollbars, the original Windows has the user click two arrow buttons to move the page up or down. The user can also click and drag the elevator to be taken to the appropriate place in the document.

ABOUT

Windows 1.0 was the first version of the Microsoft Windows line. The development of Windows began after Bill Gates, founder of Microsoft, saw a demonstration at the COMDEX 1982 of VisiCorp’s Visi On, a graphical user interface software for IBM PC compatible computers. Windows was released to the public on November 10, 1983, required two floppy disk drives and 192 KB of RAM.

ABOUT

Windows XP is a personal computer operating system produced that was released in 2001. Development for Windows XP started in the late 1990s. The system was well received as critics noted increased performance and stability, more intuitive user interface, improved hardware support, and expanded multimedia capabilities. Even though support for Windows XP ended in 2014, there are still some countries in which 38% of Windows users still use Windows XP. The scroll bar was on the right side, with arrows on the top and bottom that you could click to scroll up and down. It also had a thumb that you could click and drag as well.

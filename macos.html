macOS

CREATOR
Apple Inc.
ABOUT

The latest operating system for Apple’s desktop and server operating system was introduced in 2001 called Mac OS X 10.0 (now named macOS). Since then there have been many different updates. The Mac OS X 10.0 was a complete change from the classic Mac OS. It introduced a brand new code base, introduced new Darwin Unix-like core, and completely new system of memory management. The scroll bars had up and down arrows at the top and bottom that you could click to move scroll up and down. It also had a thumb that you could drag up and down.

HOW TO USE THE SCROLLBAR

The scrollbar has up and down arrows attached to the top and bottom of the bar. A user can click and drag on these to control the direction and distance of the scroll. Along with dragging and dropping in the system. Clicking on the "cable anchors", located at the top and bottom, let the user go to the beginning or end of the document. Originally, clicking and holding down the mouse button on the arrow keys repeated the opperation, and the cursor transported automatically with the moving elevator. Unfortunately modern browsers do not allow the cursor to be move programatically so this was not implemented on this version.

ABOUT

OPENLOOK was one of the first standard graphical user interfaces for Unix that was defined in the late 1980s by Sun Microsystems and AT&T. It was created at a time when there was little or no standardization in Unix graphical user interfaces (GUIs); the X Window System was emerging as the likely de facto standard for Unix graphical displays, but its designers had deliberately chosen not to specify any look and feel guidelines, leaving this up to application and window manager developers.
